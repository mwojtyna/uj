* Opis:
============
Program realizuje problem wzajemnego wykluczania wątków przy użyciu mutexów.

* Zawartość:
============
Makefile

* Uruchamianie:
===============
Aby uruchomić program:
$ make run

Usuwanie plików *.o, *.x:
$ make clean

Pakowanie do archiwum:
$ make tar
